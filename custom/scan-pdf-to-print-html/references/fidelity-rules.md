# Fidelity Rules

Use these rules whenever the source is a scanned or image-only PDF.

## Non-negotiable

- Do not summarize.
- Do not simplify.
- Do not add teaching notes.
- Do not silently fix perceived mistakes in the source.
- Do not normalize style in ways that change meaning.

## Preserve Exactly

- heading levels and order
- question numbering and sub-question numbering
- answer labels and option labels
- formulas and symbols
- table row and column order
- figure labels and references
- page-local reading order

## Allowed Changes

- typography
- page layout
- margins
- figure placement
- color and styling
- replacing unreadable figure screenshots with faithful redraws

## Uncertainty Marking

If a source fragment is unreadable, mark it explicitly.

Examples:

- `[TO VERIFY: denominator unclear]`
- `[TO VERIFY: second character may be x or chi]`
- `[TO VERIFY: table header cropped in source]`

Keep uncertainty local. Do not mark a whole page as uncertain when only one symbol is unclear.
