=== Tool Test: BASH ===
Tool used: "tool":"bash" (with tool.execute.before hook)
Output:
  位于分支 main
  您们的分支与上游分支 'origin/main' 一致。
  无文件要提交，工作区干净
Analysis: bash hook rewrite works correctly on macOS (no Unix-path issue unlike Windows).
  The isUnixPath guard (/^\/[a-zA-Z]\//) does NOT trigger because macOS paths
  like /Users/dht/.cargo/bin/lean-ctx are valid Unix paths that the shell can execute.

=== Conclusion ===
The same plugin file works on both Windows and macOS without modification.
All 4 tools verified working on macOS with lean-ctx compression active.
